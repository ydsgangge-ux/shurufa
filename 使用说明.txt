AI IME 输入法 - 使用说明 v1.2.0
================================

一、基本输入
-----------
- 拼音输入：直接打拼音，数字键选词，空格上屏第一个候选
- 双拼：可在配置中切换双拼方案
- 模糊音：zh/z、ch/c、sh/s 等可配置模糊

二、V键符号面板（智能ABC风格）
-----------------------------
输入 v + 数字键(1-9) 进入符号面板：

  v1 = 常用表情 Emoji    (😀😂👍❤️🔥...)
  v2 = 数字序号          (①②③ ⅠⅡⅢ...)
  v3 = 数学符号          (±×÷ ∫∑√...)
  v4 = 希腊字母          (αβγδε...Ω)
  v5 = 单位/货币          (℃￥€£ ㎡㎏...)
  v6 = 箭头/指向          (→←↑↓ ⇒⇐...)
  v7 = 勾叉/星/图形       (✓✗ ★☆ ●○...)
  v8 = 中文标点/特殊      (…— 「」【】...)
  v9 = 制表符/方块        (─│ ┌┐ └┘...)

面板内操作：
  1-9, 0 = 选择对应符号上屏
  空格   = 上屏第一个符号
  = / -  = 下一页 / 上一页
  ESC/回车 = 退出面板
  退格   = 退出面板并退回上一字符

三、AI 续写预测
--------------
- 输入法会根据你的用词习惯，在上屏一个词后自动预测下一个词
- 纯查表，零延迟，不依赖AI模型
- 查不到时自动调用AI模型兜底（需配置）

四、AI 扩词库（大模型生成领域词汇）
---------------------------------

【推荐】一键启动方式：
  双击 expand_lexicon_launcher.bat
  → 出现菜单，按数字/字母选择领域，自动扩词+重启

【命令行方式】（需先打开终端）：
  打开终端方法：
    1. 在安装目录按住 Shift + 右键 → 选择"在此处打开 PowerShell"
    2. 或：开始菜单 → Windows PowerShell → cd 到安装目录

  python expand_lexicon.py it         # IT/计算机领域
  python expand_lexicon.py 医学       # 医学领域
  python expand_lexicon.py 线束制造    # 自定义领域（LLM生成）
  python expand_lexicon.py --list     # 查看所有内置领域
  python expand_lexicon.py --all      # 导入所有内置领域

内置领域：it 编程 计算机 财经 金融 成语 地名 地理 历史名人
         人名 诗词 古诗文 医学 医疗 饮食 美食 法律 汽车 动物

非内置领域会自动调用云端LLM生成专业词汇。
THUOCL词库文件请从 http://thuocl.thunlp.org/ 下载，
放入 data\thuocl\ 目录。

五、搜狗词库导入
---------------
  python import_sogou_dict.py 词库文件.scel

从搜狗词库网站下载 .scel 文件，直接导入合并到基础词库。

六、重启输入法
-------------
修改词库或配置后，需要重启输入法生效：

  restart_ime.bat    (双击运行)

七、大模型与API配置
------------------
1. 本地模型：
   - 将 .gguf 模型文件放入 models\ 目录
   - 运行 download_model.ps1 可一键下载 Qwen2.5-0.5B
   - 支持 GPU 加速（修改 local_llm_server.py 启动参数）

2. 云端API：
   - 编辑 C:\Users\<用户名>\.ai_ime\api_config.json
   - 支持 DeepSeek、Qwen 等云端模型
   - 首次安装会自动生成 api_config.example.json 模板

八、快捷键
---------
  Ctrl+Space   切换中/英文
  Shift        临时英文模式
  Ctrl+Shift   切换输入法
